Step 1：建立 Google Sheets

前往 sheets.google.com，建立一個新的空白試算表，命名為「LINE選品評估_v13」。記下試算表的網址（裡面有一串 ID 會用到）。

Step 2：建立 Google Apps Script

在試算表點上方選單「擴充功能」→「Apps Script」，刪除原有內容，貼上以下程式碼：

// LINE 選品評估系統 v13 — Google Apps Script 後端
// 貼到 Apps Script 後，記得把下面的 SHEET_ID 換成你的試算表 ID

const SHEET_ID = '你的試算表ID'; // ← 換這裡
const SHEET_NAME = 'products';

function doGet(e) {
  const action = e.parameter.action;
  if (action === 'read') return readProducts();
  if (action === 'ping') return ok({status:'ok',msg:'LINE選品v13連線成功'});
  return ok({error:'unknown action'});
}

function doPost(e) {
  const data = JSON.parse(e.postData.contents);
  if (data.action === 'write') return writeProducts(data.products);
  return ok({error:'unknown'});
}

function readProducts() {
  try {
    const ss = SpreadsheetApp.openById(SHEET_ID);
    let sheet = ss.getSheetByName(SHEET_NAME);
    if (!sheet) return ok({products:[], nextId:1});
    const data = sheet.getRange('A1').getValue();
    if (!data) return ok({products:[], nextId:1});
    return ok(JSON.parse(data));
  } catch(err) { return ok({error:err.toString()}); }
}

function writeProducts(payload) {
  try {
    const ss = SpreadsheetApp.openById(SHEET_ID);
    let sheet = ss.getSheetByName(SHEET_NAME);
    if (!sheet) sheet = ss.insertSheet(SHEET_NAME);
    sheet.getRange('A1').setValue(JSON.stringify(payload));
    return ok({status:'saved', count: payload.products?.length || 0});
  } catch(err) { return ok({error:err.toString()}); }
}

function ok(data) {
  return ContentService
    .createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}
Step 3：取得試算表 ID

試算表網址長這樣：
https://docs.google.com/spreadsheets/d/這串就是ID/edit
把紅色那串複製，貼到 Apps Script 程式碼第一行的 SHEET_ID。

Step 4：部署為 Web App

Apps Script 右上角點「部署」→「新增部署作業」→ 類型選「網頁應用程式」
設定：以我的身份執行，存取權限選「所有人」
點「部署」→「授予存取權」→ 複製「網頁應用程式網址」

Step 5：貼回工具

回到工具左側「設定」欄，把網址貼入，按「儲存連線」→「測試」，看到「✅ 連線成功」就完成了。
